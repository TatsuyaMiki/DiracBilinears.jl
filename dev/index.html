DiracBilinears

This document provides usage for DiracBilinears.

This package provides tools for computing microscopic physical quantities in relativistic quantum theory called Dirac bilinears, which are the fundamental 16 independent quantities derived from the Dirac field. By connecting to the external first-principles calculation package Quantum ESPRESSO, Wannier90, and wan2respack (RESPACK), it can evaluate

  • spatial distributions of the bilinears
  • Wannier matrix elements of the bilinears
